# 🗼 CPAI - 职业偏好 & AI 适配度测试小程序

一个专业的微信小程序（CPAI），帮助用户了解性格特点、职业偏好，并评估在 AI 时代的适应能力和职业风险。

## 📱 功能模块

### 1. 性格测试 (MBTI)
- **20 道精选题目**，评估 4 个维度 (E/I, S/N, T/F, J/P)
- **16 种人格类型**详细解析
- 基于性格的**职业推荐**

### 2. 职业偏好度测试 (霍兰德)
- **30 道专业题目**，评估 RIASEC 六维度
- **可视化图表**展示各维度得分
- 生成**霍兰德职业代码** (如 EAS、IRC 等)
- 推荐**匹配的职业方向**

### 3. AI 时代适配度测试
- **15 道前沿题目**，评估 AI 时代的核心能力
- **4 等级评估**：领航者/适应者/跟随者/风险者
- 提供**个性化发展建议**
- 推荐**AI 时代的优势职业**

### 4. AI 取代风险评估
- **10 道深度题目**，分析职业被 AI 取代的可能性
- **风险仪表盘**可视化展示
- **多维度因素分析** (重复性/创造性/情感交流等)
- 提供**提升不可替代性的策略**

### 5. AI 应用调查
- **5 道实用问题**，了解用户 AI 使用情况
- 收集**使用频率/场景/工具偏好**
- 帮助优化产品和服务

### 6. 综合测评报告
- **一站式查看所有测试结果**
- **完成度追踪**
- **综合发展建议**
- **职业推荐汇总**
- 支持**分享结果**

## 🚀 快速开始

### 1. 环境准备
- 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册 [微信小程序账号](https://mp.weixin.qq.com/)

### 2. 导入项目
1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录：`career-ai-miniprogram`
4. 填写你的 AppID (或使用测试号)
5. 点击「导入」

### 3. 配置云开发 (可选)
如需保存用户数据和测试结果：
1. 在开发者工具中点击「云开发」
2. 创建云环境
3. 在 `app.js` 中替换 `env: 'your-env-id'` 为你的环境 ID
4. 在 `project.config.json` 中替换 `appid`

### 4. 预览和发布
- 点击「预览」在手机上测试
- 测试完成后点击「上传」
- 在微信小程序后台提交审核

## 📊 测试算法说明

### MBTI 性格测试
- 4 个维度，每维度 5 题
- 计分方式：E/I, S/N, T/F, J/P 各维度独立计分
- 结果：4 字母类型 (如 INTJ、ENFP)

### 霍兰德职业测试
- 6 个维度 (R/I/A/S/E/C)，每维度 5 题
- 计分：Likert 5 点量表 (-1 到 3 分)
- 结果：前 3 个高分维度组成代码 (如 ASE、RIC)

### AI 适配度测试
- 5 个维度：学习能力/AI 使用/创造性/协作性/适应性
- 计分：4 点量表 (1-4 分)
- 结果：百分制分数 + 4 等级评估

### AI 取代风险测试
- 10 个风险维度评估
- 计分：反向计分 (高分=高风险)
- 结果：风险等级 + 风险因素分析

## 👨‍💻 开发说明

### 添加新测试
1. 在 `pages/` 目录创建新页面文件夹
2. 在 `app.json` 中注册页面
3. 在首页添加测试入口卡片
4. 在结果页添加结果展示逻辑

### 修改题目
每个测试的题目在对应页面的 `.js` 文件中，如：
```javascript
const questions = [
  {
    question: "题目内容",
    options: ["选项 A", "选项 B", "选项 C", "选项 D"],
    scores: [2, 1, 0, -1],
    type: "维度标识"
  }
];
```

### 调整样式
全局样式在 `app.wxss`，页面样式在各自页面的 `.wxss` 文件中。
